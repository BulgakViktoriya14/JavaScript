Классы:
класс Good - класс для товара с полями:название товара,тип,цена,дата выпуска и индификатора.
класс foodProduct - класс для продовольственного товара с полями: название товара, тип,цена,дата выпуска,годен до и сроки годности.
класс Shop - класс для магазина с полями: название магазина,адрес,наценка,прибыль, товары, сумма всех товаров.
класс Market - класс для рынка с полем магазины(массив из магазинов).

Методы:
sumAllGoods() - считает сумму всех товаров в магазине
addGoods(t,n) - добавляет товар t количеством n в магазин.
deleteGoods(t,n) -удаляет количество n товара t из магазина.
sellGoods(t,n) - продажа товара n количеством n.
info() - информация о магазине.
addShop(sh) - добавляет магазин в рынок.
infoShops() - информация о магазинах рынка.
